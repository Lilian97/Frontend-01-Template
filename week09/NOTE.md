- Animation

  - @keyframes定义
  - animation使用

  ```
  @keyframes mykf{
  	from{
  		background:red;
  	}
  	to{
  		background:yellow;
  	}
  }
  div{
  	animation: mykf 5s infinite;
  }
  ```

  - 属性
    - animation-name：时间曲线
    - animation-duration：动画时长
    - animation-timing-function：动画的时间曲线
    - animation-delay：动画开始前的延迟
